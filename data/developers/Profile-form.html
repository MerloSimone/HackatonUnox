Profile | Get profile settings

  • compare changes to:
  • 0.0.0

Get logged in user informations like: name, email, generic profile info, application language, KPI and recipes unit measures (Celsius / Fahrenheit for temperatures, smc / BTU for gas, litres / oz for water and detergent). Please note that all temperatures are stored in celsius and so if programs are created using API, set temperatures must be specified in celsius
